Learning Unit 3 Divide Safely

Example: Divide Safely

File: unit3/example-DivideSafely.html

Problem Statement

Write a program that asks a user to enter two numbers. If the second number is not zero then divide the first number by the second number and output the results to the page. If the second is zero then alert the user that they entered a zero and should try again. Do not output anything to the page if the second number is zero.

The Program Planning Lists

The List of Input Variables:
    numberOne
    numberTwo
    

The List of Output Variables:
    numberOne
    numberTwo
    results
    

The Process Checklist:
    prompt user for numberOne
    prompt user for numberTwo
    check numberTwo for zero
    alert user based on value
    divide numberOne by numberTwo
    output results
