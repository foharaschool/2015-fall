Lab problem statement

Write a program that asks a user to enter a number between 1 and 100 that does not include the end points. When a number in the correct range is entered then output the number and a message indicating success. If the number is not in the range then output the number and ask the user to try again.

Lab Steps

  1. Open this file in Firefox and jEdit.
  2. Add your name, section, and email.
  3. Add code after the line: YOUR CODE STARTS AFTER THIS LINE: to complete this lab.
  4. Your output should look like the Example Output below.
  5. Variable declarations section
    1. Write a comment that identifies this coding section.
    2. Declare a variable that will be used to hold a number.
    3. Declare a variable that holds the low endpoint of the range.
    4. Declare a variable that holds the high endpoint of the range.
    5. Assign 1 and 100 to the low and high endpoint variables at the time they are defined. (Note that these range end points are constants, and thus must have variable names composed of uppercase letters with underscores to separate words in their names.)
  6. Assignments section
    1. Write a comment that identifies this coding section.
    2. Ask the user to enter a number between 1 and 100 that does not include the end points of the range and store it in the number variable.
    3. Use the length property of a string object to output an error message if the user enters nothing at all into the prompt text box. The error message should say something about their failure to enter a number, and that they should start over by refreshing the Web page.
    4. Convert the input to a number.
  7. Logic and Output section   (Note: This could be two separate sections.)
    1. Write a comment that identifies this coding section.
    2. Use only variables in your logic.
    3. If the entered number is in the specfied range then output a success statement.
    4. If the number is out of the specified range then output a request to try again.
