Part 1 Problem Statement

Write a program that asks a user for two numbers. If the first number is greater than or equal to the second number then output a statement like this, "<first number> is greater than or equal to <second number>." If the first number is less than the second number then output a statement like, "<first number> is less than <second number>." Use an if/else statement for this part.
