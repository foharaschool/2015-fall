Lab problem statement

Write a program that asks a user to enter a string. Then ask the user to enter a string that will sort after the first string in an ASCII sort. ("A" sorts before "B".) If the second string is after the first in an ASCII sort then output a message that includes both strings and a success message. If the second string does not sort after the first string then output the strings and ask the user to try again. Note that this now should be done using an ELSE with the IF, and not done using two separate IF statements. And once again, building the message into a message variable, which is then displayed as the last step of the program is a good approach to take here.

If there is extra time, after each prompt, use an IF statement interrogating the length property of the variable holding the entered string, and if the user did not enter anything at all (i.e. the value of the length property was zero), alert them with a CAUTION message of some kind.
