Problem Statement

Write a program that asks a user to enter a number greater than 10. If the number is not greater than 10 then write a comment to the page that says they did not enter a number greater than 10. In either case output the number to the page.

OPTIONAL: Once you have this Lab 01 working correctly, if there is extra time, look up the isNaN() function in your book. Then try applying the isNaN() function to the variable that receives the output from the prompt() method. This would be done immediately after the statement that invokes the prompt() method, but before the statement that invokves the Number() function. If what the user entered was not capable of being converted to a number, have the program put out an error message or two, notifying the user of their failure to enter a number, and, telling them to start over by refreshing the Web page.
