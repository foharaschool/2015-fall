Example 4_2: Print Examination Scores

File: unit4/example4_2.html

Problem Statement

Write a program to read and print a series of names and exam scores for students enrolled in a mathematics course. The class average is to be calculated and printed at the end of the report. Scores can range from 0 to 100. Read all scores from the Student Exam Records set.

Remember: a program should always be thought of as consisting of a set of functions. However, in this simple case, just to illustrate the basics of working with a function, here the set consists of only a single function. Most real programs will consist of a set of functions with a main (aka mainline aka controller) function, and one or more subfunctions.

Program Planning Lists

  • When the entire program (set of functions) consists of just a single function, one can easily still write the three planning lists based on the verbs and nouns you see in the problem statement.
