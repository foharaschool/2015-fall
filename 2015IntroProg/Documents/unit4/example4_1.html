Example: Fahrenheit-Celsius Conversion

File: unit4/example4_1.html

Problem Statement

A program is to be written that will accept five Fahrenheit temperatures, convert each one to Celsius, and display the converted temperatures on the page. After all 5 have been processed, the program will display the words, "All temperatures processed." on the page.

Remember: a program should always be thought of as consisting of a set of functions. However, in this simple case, just to illustrate the basics of working with a function, here the set consists of only a single function. Most real programs will consist of a set of functions with a main (aka mainline aka controller) function, and one or more subfunctions.

Program Planning Lists

  • When the entire program (set of functions) consists of just a single function, one can easily still write the three planning lists based on the verbs and nouns you see in the problem statement.
The List of Input Variables:
    currentFahrenheitTemperature
    

The List of Output Variables:
    currentFahrenheitTemperature
    currentCelciusTemperature
    finalMessage

The Process Checklist:
    prompt user for 5 temperatures in a loop
    - convert the inputs to numbers
    - display each input temperature
    - display each converted temperature
    display finalMessage

The Program OverView Statement:


1. Overall, this program processes data about:  Temperatures

2. Overall, this program obtains its input from:    the viewer

3. Overall, this program produces as output: Celcius temperature equivalents of Fahrenheit temperatures

4. Overall, this program delivers this output to:   the Web page
