Part 1 - Problem Statement

Write a program that loops 10 times maximum.  Within the loop, ask the user to enter a number. If the entered value is a number then add the number to a total. If the user enters a value that is not a number then alert them that they didn't enter a number and stop the looping. In any event, the program must display on the Web page the number of times that the loop did run, and, then the sum of all entered numbers.   [Note: To determine if the user entered a number or not, use the isNaN() function.]

Before begining, your team must set up a version plan to guide you when you program.

 

Part 1 Sample Output 

The number of loops: 10
The total: 594
