Part 1 Problem Statement

Write a program that outputs the numbers 1 through 50. Next to each number, output the value that is half of the number, and then output the value that is one quarter of the number. Format the output as in the sample output. Use a for loop for this part.

1, 0.5, 0.25
2, 1, 0.5
3, 1.5, 0.75
4, 2, 1
5, 2.5, 1.25
...
