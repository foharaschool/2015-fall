Lab Steps

  1. Open this HTML file in Firefox and jEdit.
  2. Add your name, section, and email.
  3. Open the file /unit4/jsFiles/lab03firstWhileLoop.js in jEdit.
  4. Create a JavaScript program that will ask a user to enter five instances of anythng they wish to key in, with each such string of characters displayed on the web page. See sample output below.
  5. Variable declarations section
    1. Write a comment that identifies this coding section.
    2. Declare a constant with a name something like NUMBER_TIMES_TO_LOOP, and assign it a value of 5.   (Note: when constants are set up, IAW course standards, they are assigned a value right on the var statement itself. Also note that IAW course standards, they are not named using camel case format.)
    3. Declare a variable to be used to control the looping. A name such as loopCounter would make sense.
    4. Declare a variable to hold the string entered by the user.  A name such as userEnteredString for example would be a good name.
  6. Assignments section
    1. Write a comment that identifies this coding section.
    2. Assign the loopCounter variable the value zero (0).  (Note: the zero can be in a constant named ZERO, or can just be a literal, since this zero really just means zero, and does not have any other "secret" meaning in the context of this application.)
  7. Processing Section
    1. Write a comment that identifies this coding section.
    2. Use a while loop and the loop counter variable.
    3. In the while boolean expression, test if the loop counter variable is less than 5.
    4. In the loop you should:
      1. Prompt the user for an entry and assign it to the variable set up above to hold the string entered by the user.
      2. Display what the user entered on the page.
      3. Increment the loop counter variable.
