The Problem Statement

Write a program that asks a user to enter a number. If the entered number is greater than a starting variable that has been initialized to zero, then output the number and the value in the starting variable, and then store the entered number into the starting variable, and ask the user to enter a bigger number. As long as the newly entered number is bigger than the last one, then keep asking the user to enter another bigger number. Display each number that is entered as shown in the sample output below. Once the user enters a number that is smaller than the last one entered then display the sign off message "Good bye!"

Remember: Always begin by analyzing the problem statement. In this exampe, even if one assumes that a number will always be entered, the above problem statement as written leaves a glaring question unanswered.
