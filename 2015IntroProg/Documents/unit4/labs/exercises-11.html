Part 1 Problem Statement

Write a program that accesses the Student Exam Records Set. Create a report for the instructor that calculates how many students received each grade based on the following table.

Exam ScoreAssigned Grade
90 and aboveA
80 to less than 90B
70 to less than 80C
60 to less than 70D
below 60F

Also calculate the average grade for the class. Output the report as in the sample output.

  • Sample Part 1 Output:
Grade		Count
A		4
B		6
C		5
D		5
F		3

Average Score: 74.90434782608696

Part 2 Problem Statement

Write a guessing game program. Ask a user to guess a number between 1 and 100. If a guess is not a number then alert the user. If the number is too big then alert the user and output the guess to the page. If the number is too small then alert the user and output the guess to the page. When the user guesses the right number then alert the user that they guessed right and output the final guess to the page.

The correct value that is being guessed should be calculated with the following code:

  • Sample Part 2 Output:
small guess: 50
small guess: 70
large guess: 80
large guess: 78

Correct guess: 75
