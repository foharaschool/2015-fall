jsLint Overview Review

  • When you go to the jsLint site you an area to enter your code and a settings area with a lot of check boxes.
  • Here are the steps to using jsLint.
    1. Access the site at http://www.jslint.com/
    2. Set the checkboxes in the settings area.
    3. Copy and paste your code into the top text area.
    4. Click on the button labelled jsLint.
    5. Fix errors and issues that jsLint finds.
    6. When you no longer get an errors box then you are done!
