Arrays - A More Flexible Data Storage System

  • All modern programming languages have ways to store data in variables that are flexible, adaptable, easy to use, and very similar to every other programming language.
  • The first of these storage systems we are going to learn is called Arrays.
  • This line of code creates a single variable and assigns one piece of data to it.
  • Here is an array of zip codes. The code is creating a variable and assigning an empty array to it.
  • We use a set of square brackets ([]), that does not have a space between them, to make a new array.
  • What does this really create? Well, an Array is an object. You will learn lots more about objects in the coming semesters. For now, think of an Array as a piece of running JavaScript code that lets us add data to it.
