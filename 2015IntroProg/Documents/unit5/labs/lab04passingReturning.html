Problem Statement

Design a program that will prompt a user for their first name, middle initial, and last name. Once the program has these three strings, then have it display the user's name on the page with this format: "Hello, John T. Public. How are you?"

  1. Lab Details
    1. Write the Program Planning documents in comments in the JavaScript file.
    2. Use the prompt method for user input.
    3. Use a function to format the full name. Name the new function something like: formatFullName()  or  formatUserFullName(). The function must have 3 parameters: one for the first name, one for the middle initial, and one for the last name. The function should return a string that is the full name with this format, including period for the middle initial:
      • "Bob T. Smith"
    4. The main function should prompt user the user for the three parts of his/her name. It must then call the name formatting function (see Step #3 just above). The main function must then then display on the web page the full name with some additonal greeting text around it such as the greeting text shown in the example.
    5. You will need to add some HTML elements to this Web page, either directly or with JavaScript.
