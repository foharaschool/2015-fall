Lab Steps:

   A.  In the HTML File:

  1. Add a form to this HTML document between the comments. Use the form in /unit5/usingForms.html as a starting place.
  2. The form should have four input text fields:
    • A field for an address
    • A field for a city
    • A field for a State
    • A field for a zip code
  3. The form should have a submit button.
  4. The opening form tag should have an onsubmit= attribute (aka event) that calls a function named displayformattedAddress. There must be a return clause like the example, since the called function will be returning a Boolean value of false.  [ Important Note: Unlike an onclick= attribute (aka event) that will NOT accept a Boolean value even if you tried to return one to it, the onsubmit= attribute (aka event) must have a Boolean value of false returned to it to prevent the source code files associated with the web page from being refreshed, and thus losing any materials that were placed into them by the JavaScript program. ]
  5. Just below the form, set up an output HTML element (e.g. a div element) in the HTML page. Give it an id attribute so the JavaScript function can access it.

   B.  In the javaScript File:

  1. Create a function in the JavaScript file named displayformattedAddress().
  2. Version one of this (or any new function) is always nothing but an invoking of the alert() method, with a message something like "The FOC entered the new function."  DO NOT SKIP THIS STEP.
  3. Your function will get the values of all the input fields on the form (and leave these strings in four appropriately named variables, such as keyedAddress, keyedCity, etc.).
  4. The function will create a formatted address output something like this:

    123 Some St.
    Madison, WI 53704
  5. Output the formatted address to the output HTML element by using the innerHTML property.
  6. Does the function need to have a  return false;  statement as the last instruction (to make sure no browser refreshes the web page following the successful submit--thus wiping out what we just placed into it!?)
