Part 2 Problem Statement

Write a program that asks a user to enter a string. Output the following for the entered string.

  • the original string.
  • the length of the string with: string.length.
  • the string in all uppercase letters with: string.toUpperCase().
  • the string in all lowercase letters with: string.toLowerCase().
  • the location of the first space in the string or the string "no spaces" with: string.indexOf(" "). This functions returns -1 of the search string is not present.

Use a function to perform the calculations and return the formatted output string. Format the output with HTML tags.
