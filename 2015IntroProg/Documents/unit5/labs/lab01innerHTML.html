Lab Steps

Note: This is an execise just to become familiar with the technical aspects of using the  document.getElementById() method in conjunction with the  innerHTML property of HTML element objects, in order to insert content from within a JavaScript program onto a web page.

And remember: work in 'versions' testing as you go. Do not 'bury' mistakes, and then after many more lines of code have been added, attempt to find what is wrong. By that point, there will be too many possibilities to readily find the error(s).

  1. Open this file in Firefox and jEdit.
  2. Add your name, section, and email.
  3. Open the JavaScript library file /unit5/jsFiles/lab01innerHTML.js  in jEdit.
  4. Open and enter the HTML file for this lab ( /unit5/labs/lab01innerHTML.html) and find the spot where there is an HTML comment that says, "Your HTML code after this!".
  5. Add some HTML below the comment but above the <script> tag.
    1. Add an HTML table element with an id attribute.
    2. Add a header row using a couple of <th> tags.
    3. Don't forget the </table>!
    4. You won't need to insert any rows in the table at this point; we'll be doing that with JavaScript.
  6. Now switch to the JavaScript library file, /unit5/jsFiles/lab01innerHTML.js.
  7. First, remove the document.write() call from the function.
  8. Add the following to the function.
    1.  A variable that will hold (a reference to) the table element. A good name for this variable might be: "myTable".
    2.  A variable that will hold (a reference to) some rows to (later) insert into the table. A good name for this variable might be "someRows".
  9. Get (a reference to) the table element and assign it to the variable created in 8.A. above. Use the document.getElementById() method for this.
  10. Build up (just using an assign operator) the 'string' variable created in 8.B. above with some HTML rows and cells. You will need to include the tags and the data in the the strings. Make 4 or five rows.
  11. Append the 'string' variable (containing the rows set up in the previous step) to the innerHTML attribute of the table element. (How do you append to an existing string? I.e. you don't want to erase the already existing content of the table, but rather wish to place more on the end of what is already there).)
  12. Refresh the page one last time (to execute the program) to make sure the program is working as it should..
