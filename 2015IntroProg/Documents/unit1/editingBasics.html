Let's start with a live demo of the editor we are using this semester. Here's the demo page: Demo: jEdit

Here are the steps

  • Run your version of Firefox from your external hard drive.
  • Run your version of jEdit from your external hard drive.
  • Find and open the lab or project file that you need to modify with jEdit. This can be done in any way you find convenient. For example, you could use a drop and drag technique. You could use the "File" item on the menu. Or, you could use the File Browser feature of jEdit itself, as shown below.

  • Put your name and section in the documentation section of the file.

  • Write your code.

  • Save the lab or project file.
  • Use Firefox to open or refresh the page and see the results.
