Using Learning Units

About Learning Units

  • Learning Units are a big part of the course. They are a complete little Web site that you download. They contain learning materials, project files, lab files, demos, and link to supplementary screencasts.
  • They have folder/directories for demos, labs, the project, and resources like images and css files.
  • The lecture notes and learning activities are accessible from the learning unit's "home page" named index.html.
  • We will download a learning unit compressed (aka zipped) file from Blackboard when we start on that unit.  To accomplish this, follow the directions in the "Getting Started ...." booklet.
  • The downloaded file is then decompressed (aka unzipped.)  To accomplish this, follow the directions in the "Getting Started ...." booklet.
  • You will be editing the project and lab files that are already part of the learning unit.
  • Then you compress (aka zip) the entire learning unit back up and upload it to Blackboard for evaluation.  To accomplish this, follow the directions in the "Getting Started ...." booklet.

Navigation note:  At the bottom of each page is a link in the form of a button with the word "Back" on it.  Clicking this is the same as clicking the browser's Back button, or, using the [Alt + left arrow] combination.
